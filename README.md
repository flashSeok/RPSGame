# RPSGame (가위바위보 게임)
### 학습내용
#### UIKit
* UILabel
  * 열거형의 rawValue값을 변수에 저장 후 switch문으로 각 변수마다 값 설정 

* UIButton
  * 버튼 클릭시 .currentTitle 메서드를 통해서 선택한 버튼의 값을 알 수 있음.(액션함수 내 여러 버튼이 있을 때 어느 버튼을 눌렀는지 확인 가능)
